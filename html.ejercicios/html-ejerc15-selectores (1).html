Tipos de selectores que entran en examen

Clase (class=””)

Este es un ejemplo de un párrafo con la clase "ejemplo-clase". El texto de este párrafo se mostrará en color azul.

El selector de clase se utiliza para aplicar estilos a elementos que comparten la misma clase. Se puede usar para aplicar estilos a varios elementos a la vez sin importar su tipo.

Identificador (id=””)

Este es un ejemplo de un párrafo con el id "ejemplo-id". El texto de este párrafo se mostrará en color rojo.

El selector de identificador se utiliza para aplicar estilos a un único elemento que tenga un id específico. Los identificadores deben ser únicos en una página HTML.

Todos los elementos (*)

Este párrafo usa el selector universal (*) para aplicar la fuente "Arial" a todos los elementos de la página.

El selector universal (*) aplica el estilo a todos los elementos dentro del documento HTML.

Varios elementos (p, em, h1, h2)

En este caso, se aplicaron estilos a los elementos p, em, h1 y h2 para hacer que su texto aparezca en cursiva.

Elementos contenidos en otros elementos (ol li)

El selector de elementos contenidos en otros elementos aplica estilos a los li dentro de una lista ordenada ol. En este caso, los elementos de la lista se muestran en color verde.

Sobrevuleo (p:hover)

Coloca el cursor sobre este párrafo para ver cómo cambia de color al pasar el mouse sobre él.

El selector hover se utiliza para aplicar estilos cuando el usuario pasa el mouse sobre un elemento. En este caso, el color del texto cambiará a morado cuando pases el ratón sobre el párrafo.
